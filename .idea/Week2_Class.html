Bootstrap buttons

Bootstrap can style and color-code buttons.

The default colors aim to reflect the meaning of the buttons. It's up to you to decide on the meaning of the buttons in your app.

So buttons for dangerous actions (e.g. delete your account) are red, warnings are orange.

Regular buttons are white/grey, info buttons are blue

Primary buttons, indicating the most likely flow for the application, are blue. Positive buttons are green

None of the buttons do anything yet, other than look nice. We'll wire up some button functionality soon!

A dropdown menu displays a list of links that expand from a button. Links can be on the same page, same site, or external. Bootstrap created this dropdown from a HTML button element, and an unordered list of a link elements.

Select an option from the dropdown menu to go to Wikipedia page for that animal.
